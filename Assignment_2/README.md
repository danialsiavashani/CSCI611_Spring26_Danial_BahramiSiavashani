
# Assignment 2 - Image Filtering and CNN

## Requirements
- Python 3.x
- PyTorch
- OpenCV (opencv-python)
- NumPy
- Matplotlib

Install dependencies:
pip install torch torchvision opencv-python numpy matplotlib

## How to Run

### Part 1 - Image Filtering
Open and run image_filtering.ipynb in Jupyter Notebook.
The notebook uses building2.jpg and car.jpg included in the folder.

### Part 2 - Build CNN
Open and run build_cnn.ipynb in Jupyter Notebook.
The CIFAR-10 dataset will be downloaded automatically on first run.
